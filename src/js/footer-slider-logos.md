# Footer Logos Slider

Модуль для управления слайдером логотипов партнёров в подвале сайта.

## Описание

Этот модуль реализует функциональность переключения между группами логотипов партнёров с плавной анимацией. Логотипы отображаются группами по 6 штук, с возможностью навигации между группами с помощью кнопок "назад" и "вперёд", а также индикатора пагинации.

## Основные возможности

- **Автоматическое скрытие элементов**: Если логотипов 6 или меньше, кнопки навигации и индикатор пагинации автоматически скрываются
- **Плавная анимация листания**: Переключение между группами логотипов происходит с эффектом горизонтального скольжения (fade + translateX)
- **Фиксированная высота контейнера**: Контейнер имеет минимальную высоту, что предотвращает "прыжки" контента при переключении
- **Направленная анимация**: Логотипы выезжают/въезжают в зависимости от направления навигации (вперёд/назад)
- **Бесконечная прокрутка**: Навигация циклическая - после последней группы показывается первая и наоборот
- **Адаптивное количество**: Если в последней группе осталось меньше 6 логотипов, показываются все оставшиеся
- **Защита от множественных кликов**: Блокировка повторных кликов во время анимации
- **Индикатор пагинации**: Визуальные точки показывают текущую страницу и позволяют переходить напрямую к любой странице
- **Интерактивность**: Точки пагинации имеют hover-эффект и анимацию при переключении

## Принцип работы

1. **Инициализация**: При загрузке страницы модуль находит контейнер с логотипами, кнопки и индикатор пагинации
2. **Подсчёт логотипов**: Определяется общее количество логотипов и количество страниц
3. **Создание пагинации**: Генерируются точки-индикаторы для каждой страницы
4. **Настройка**: Каждому логотипу добавляются необходимые стили и атрибуты для анимации
5. **Отображение**: Показывается первая группа из 6 логотипов, первая точка помечается как активная
6. **Навигация**:
   - При клике на кнопки происходит переключение на следующую/предыдущую группу
   - При клике на точку происходит переход к соответствующей странице
   - При каждом переключении обновляется активная точка

## Параметры

- `logosPerPage`: Количество логотипов на одной странице (по умолчанию 6)
- `currentPage`: Индекс текущей отображаемой страницы
- `totalPages`: Общее количество страниц
- `isAnimating`: Флаг для предотвращения множественных кликов во время анимации

## Анимация

Анимация реализована через CSS transitions с использованием `transform: translateX()`:

- **Opacity**: Плавное появление/исчезновение (0.5s)
- **Transform translateX**: Горизонтальное скольжение на 30px (0.5s)
- **Timing**: Cubic-bezier(0.4, 0, 0.2, 1) для плавности и естественности движения
- **Направление**:
  - При навигации вперёд: старые элементы уезжают влево, новые приезжают справа
  - При навигации назад: старые элементы уезжают вправо, новые приезжают слева
- **Фиксированная высота**: Контейнер имеет `min-height: 280px` и `overflow: hidden` для предотвращения смещения контента

## Использование

Модуль автоматически инициализируется при загрузке DOM:

```javascript
document.addEventListener('DOMContentLoaded', () => {
  new FooterLogosSlider();
});
```

Для работы модуля необходимо:

1. Подключить скрипт в HTML
2. Убедиться, что в DOM присутствуют элементы с классами:
   - `.footer__content__partners__items`
   - `.footer__content__partners__buttons`
